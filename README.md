# mongodb-s3-backup

## Run
```
docker run -e MONGODB_HOST mongodb://localhost -e AWS_SECRET_ACCESS_KEY abc -e AWS_ACCESS_KEY_ID xyz -e S3_BUCKET bucket-name no0dles/mongodb-s3-backup
```

## Environments
| Name | Description |
| -- | -- |
| MONGODB_HOST | asd |
| AWS_SECRET_ACCESS_KEY | asd |
| AWS_ACCESS_KEY_ID | asd |
| S3_BUCKET | asd |
